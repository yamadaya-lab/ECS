9. 縦軸を対数目盛りで表示することも出来る。 次のset命令で縦軸を対数モードにする。

set log y

ここで、plot [-1:1][0.1:10] exp(x) を実行するとグラフは直線で表示される。

確かに縦軸は対数目盛りだ。
応用問題：表示された直線の傾きはいくらか。

対数方眼紙のように縦軸の目盛を方眼表示させてみる。
縦軸の小目盛も方眼表示するには、次のようにモードセットする。

set grid ytics mytics
replot

対数モードから通常の線形(リニア)モードに戻すには、

set nolog y
plot [-1:1] exp(x)

フーリエ解析した結果の表示など、横軸を対数目盛にしたい場合は

set log x とすればよい。

TIPS

グラフウィンドウをアクティブにして、'l' を押しても対数メモリモード（縦軸のみ）の On/Off が切り替わる。
