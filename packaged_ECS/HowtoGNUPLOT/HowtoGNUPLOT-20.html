17. では値のついていない小目盛はどのように設定するのだろう。

xtics に加えて、mxtics と言うモード設定が用意されている。

set xtics 0,5,10
set mxtics 5
plot [0:10] x

を実行してみよう。（次の例は、小目盛を表示するために大きく取り込んである）

0,5,10の間に小さな目盛が見える。上記の指定により主目盛間を５等分するように小目盛が描かれた。
縦軸には mytics と言うモード設定を同様に行えばよい。
また方眼を小目盛に対しても描くには

set grid xtics mxtics ytics
replot

の様に、方眼を描きたい軸を逐一指定すると良い。
