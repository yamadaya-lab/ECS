使い方 How to use
    ・とにかくプロット plot x
    ・横軸の描画範囲の指定 plot [AAA:BBB] x
    ・縦軸の描画範囲の指定 plot [AAA:BBB][CCC:DDD] x
    ・方眼を入れる set grid
    ・方眼表示をやめる set nogrid
    ・複数のグラフを同時描画 plot x,2*x
    ・描画範囲指定+複数のグラフ描画・応用 plot [AAA:BBB] x,2*x
    ・初等関数の描画・練習問題 plot sin(x), exp(x)
    ・対数目盛 set log y
    ・データプロットにチャレンジ
    ・ローレンツアトラクタをプロットしてみる plot "AAA.dat" using 1:3[使用するデータ列の指定]
    ・縦･横軸に見出し(label)をつける set xlabel "AAA" [必須]
    ・３次元空間内に軌道を描画 splot "AAA.dat" using 2:3:4[使用するデータ列の指定]
    ・２次元曲面の表示
    ・データ＋数学関数の同時描画 plot "Lorenz.dat",sin(x)
    ・凡例(title)の設定 plot x title "y=x" [必須]
    ・横軸の目盛の設定 set xtics AAA,BBB,CCC
    ・小目盛の設定と方眼を入れる set mxtics AAA
    ・任意の位置に目盛を入れる set xtics("AAA" XXX,"BBB" YYY,...)
    ・表示をクリップボードに取り込む (Win版のみ）Wordなどに貼り付けるのに便利
    ・ディレクトリを扱うコマンド

TIPS

  • バージョン3.8以上では、マウス操作のサポートが行われ、2次元グラフでの拡大や3次元グラフでの視点移動が可能になっています。
    また、ホットキーとして、グラフ画面上で'h' :ヘルプ、'm' :マウス機能 ON/OFF、 'g' :グリッド、'l' :対数軸、'e' : replot などの操作も可能になっています。

    これらの機能追加にともない、グラフィック画面での"線種の指定"や"コピー"などのオプション操作を行う場合は，マウス機能をOFF ('m') してから行う必要があります。
