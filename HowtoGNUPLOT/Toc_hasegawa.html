使い方 How to use

  • とにかくプロット Simple plot at first plot x
  • プロットと実線を引くSimple plot at first plot x with line
  • 横軸の描画範囲の指定 Range of abscissa plot [AAA:BBB] x
  • 縦軸の描画範囲の指定 Range of ordinate plot [AAA:BBB][CCC:DDD] x
  • 方眼を入れる graph paper mode set grid
  • 方眼表示をやめる eliminating grid set nogrid
  • 複数のグラフを同時描画 plotting some graphs plot x,2*x
  • 描画範囲を指定して複数のグラフを描く・応用問題 to plot some graphs in given area. plot [AAA:BBB] x,2*x
  • 初等関数の描画・練習問題 To plot elemental functions plot sin(x)
  • 対数目盛 logarithmic scale set log y
  • データプロットにチャレンジ data plot
  • ローレンツアトラクタをプロットしてみる To plot Lorenz attractor plot "Lorenz.dat"
  • 縦･横軸に見出し(label)をつける To put axis label set xlabel "AAA"
  • ３次元空間内に軌道を描画 Displaying trajectories in 3-dim. space splot "Lorenz.dat" using 2:3:4
  • ２次元曲面の表示
  • データファイルと数学関数のグラフを同時に描画 To plot data and function graph plot "Lorenz.dat",sin(x)
  • 凡例(title)の設定 To set title of each graph plot x title "y=x"
  • 横軸の目盛の設定 To set ticks mode of abscissa set xtics AAA,BBB,CCC
  • 小目盛の設定と方眼を入れる Setting small ticks and grid set mxtics AAA
  • 任意の位置に目盛を入れる To put tick at arbitral position set xtics("AAA" XXX,"BBB" YYY,...)
  • 表示をクリップボードに取り込む How to put graph into clipboard
  • ディレクトリを扱うコマンド How to change directory, etc.
